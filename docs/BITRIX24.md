# Integração Bitrix24 - Fast Grava ES

## Configuração
1. Obter webhook URL no Bitrix24
2. Configurar em /settings/integrations
3. Mapear campos

## Sincronização
- Deals → Jobs
- Companies → Clientes
- Contacts → Contatos

## Webhook Events
- deal.add: Novo job criado
- deal.update: Job atualizado
