# Sistema de Calendário

## Visão Geral

Calendário de produção com visões diária e semanal.

## Páginas

- **DailyCalendar**: Visão diária
- **WeeklyCalendar**: Visão semanal

## Funcionalidades

- Drag & drop de jobs
- Cores por status
- Filtros por máquina
- Integração com Kanban

## Componentes

- TodayTimeline
- CompactTimeline
- JobCard

## API

```typescript
useCalendarEvents()
useScheduling()
```
