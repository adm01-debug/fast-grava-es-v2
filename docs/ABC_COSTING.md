# ABC Costing (Activity-Based Costing)

## Visão Geral
Sistema de custeio baseado em atividades para gravação a laser.

## Componentes
- ABCActivityRatesCard
- ABCCostBreakdownChart
- ABCCostPoolsCard
- ABCJobCostsTable
- ABCTechniqueChart

## Hooks
- useABCCalculations
- useABCCosts
- useABCData
- useABCMutations

## Cálculos
```
Custo Total = Σ(Atividade × Taxa × Tempo)
```
