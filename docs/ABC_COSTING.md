# ABC Costing (Activity-Based Costing)

## Visão Geral

Sistema de custeio baseado em atividades para gravação a laser.

## Componentes

- **ABCActivityRatesCard**: Taxas por atividade
- **ABCCostBreakdownChart**: Gráfico de custos
- **ABCCostPoolsCard**: Pools de custo
- **ABCJobCostsTable**: Custos por job
- **ABCTechniqueChart**: Comparação técnicas

## Cálculos

```
Custo Total = Σ(Atividade × Taxa × Tempo)
```

## API

```typescript
useABCCalculations()
useABCCosts()
useABCData()
```
